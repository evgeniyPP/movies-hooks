### Приложение, написанное на React, с использованием React Hooks и IMDb API. <br>
Позволяет искать фильмы по названию и смотреть их подробное описание. <br>
Full responsive. <br>
Сделан полностью самостоятельно (идея была взята из статьи на Medium, но 90% кода было написано "из головы") <br>

### Посмотреть: https://epp-movies-hooks.herokuapp.com/
